Principais formatações

Negrito / Destaque

Nessa frase temos um termo negrito usando a tag B (Não semântica).

Nessa frase, tenho um termo em destaque usando a tag STRONG (Semântico).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (Não semântica).

Nesta frase temos um termo em ênfase usando a tag EM (Semântico)

Texto marcado

Podemos criar um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo. Usando as tags BIG e SMALL.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado. Usando a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usando a tag INS.

Existe também o sublinhado com a tag U (Não semântica).

Teto sobrescrito

Para inserir coisas do tipo x20+3 usamos a tag SUP.

Texto subscrito

Para inserir coisas do tipo H20 usamos a tag SUB.
